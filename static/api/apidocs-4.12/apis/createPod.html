Apache CloudStack 4.12.0.0 Root Admin API Reference

createPod

Creates a new Pod.

Request parameters

Parameter NameDescriptionRequired
gatewaythe gateway for the Podtrue
namethe name of the Podtrue
netmaskthe netmask for the Podtrue
startipthe starting IP address for the Podtrue
zoneidthe Zone ID in which the Pod will be createdtrue
allocationstateAllocation state of this Pod for allocation of new resourcesfalse
endipthe ending IP address for the Podfalse
