Apache CloudStack 4.12.0.0 Root Admin API Reference

updateAccount

Updates account information for the authenticated user

Request parameters

Parameter NameDescriptionRequired
accountCurrent account namefalse
accountdetailsDetails for the account used to store specific parametersfalse
domainidThe UUID of the domain where the account existsfalse
idAccount UUIDfalse
networkdomainNetwork domain for the account's networks; empty string will update domainName with NULL valuefalse
newnameNew name for the accountfalse
roleidThe UUID of the dynamic role to set for the accountfalse
