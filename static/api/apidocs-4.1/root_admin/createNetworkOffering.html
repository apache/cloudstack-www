createNetworkOffering

Creates a network offering.

Parameter NameDescriptionRequired
displaytextthe display text of the network offeringtrue

supportedservicesservices supported by the network offeringtrue

availabilitythe availability of network offering. Default value is Optionalfalse

ispersistenttrue if network offering supports persistent networks; defaulted to false if not specifiedfalse

serviceofferingidthe service offering ID used by virtual router providerfalse

specifyvlantrue if network offering supports vlansfalse
tagsthe tags for the network offering.false
