Apache CloudStack 4.12.0.0 Root Admin API Reference

updateNuageVspDevice

Update a Nuage VSP device

Request parameters

Parameter NameDescriptionRequired
physicalnetworkidthe ID of the physical network in to which Nuage VSP is addedtrue
apiversionthe version of the API to use to communicate to Nuage VSDfalse
hostnamethe hostname of the Nuage VSDfalse
passwordthe password of CMS user in Nuage VSDfalse
portthe port to communicate to Nuage VSDfalse
retrycountthe number of retries on failure to communicate to Nuage VSDfalse
retryintervalthe time to wait after failure before retrying to communicate to Nuage VSDfalse
usernamethe user name of the CMS user in Nuage VSDfalse

Response Tags

Response NameDescription
apiversionthe version of the API to use to communicate to Nuage VSD
cmsidthe CMS ID generated by the Nuage VSD
hostnamethe hostname of the Nuage VSD
nuagedevicenamethe name of the Nuage VSP device
physicalnetworkidthe ID of the physical network to which this Nuage VSP belongs to
portthe port to communicate to Nuage VSD
providerthe service provider name corresponding to this Nuage VSP device
retrycountthe number of retries on failure to communicate to Nuage VSD
retryintervalthe time to wait after failure before retrying to communicate to Nuage VSD
vspdeviceidthe device id of the Nuage VSD
