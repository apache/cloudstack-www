Apache CloudStack 4.18.0.0 Root Admin API Reference

quotaCredits

Add +-credits to an account

Request parameters

Parameter NameDescriptionRequired
accountAccount Id for which quota credits need to be addedtrue
domainidDomain for which quota credits need to be addedtrue
valueValue of the credits to be added+, subtracted-true
min_balanceMinimum balance threshold of the accountfalse
quota_enforceAccount for which quota enforce is set to false will not be locked when there is no credit balancefalse

Response Tags

Response NameDescription
creditsthe credit deposited
currencycurrency
updated_bythe user name of the admin who updated the credits
updated_onthe account name of the admin who updated the credits
